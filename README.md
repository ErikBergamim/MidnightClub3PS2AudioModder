# MidnightClub3PS2AudioModder

.NET program that automates the conversion of WAV files for insertion into the Midnight Club 3 Dub Edition Remix game, made for ps2.

## Credits

- Tool for converting to custom WAV: [FFmpeg](https://github.com/FFmpeg/FFmpeg)
- Tool for converting to PS2 format: [PS2 Tool](https://gamebanana.com/tools/6656)
- Tool for converting to RSTM format: [rstm_build](https://github.com/EdnessP/scripts/blob/4663f469934762caf5565fffafbceb75961de02b/midnight-club/rstm_build.py#L29)
